# Borg Pharmacy WhatsApp Bot - Termux

## التثبيت

```bash
cd ~/borg-bot
npm install
node index.js
```

## التشغيل

1. افتح تطبيق Borg Pharmacy.
2. ادخل إلى صفحة **بوت واتساب**.
3. اكتب رقم البوت بدون `+` مثل: `9677xxxxxxxx`.
4. فعّل البوت واضغط **حفظ وإرسال كود الربط**.
5. شغّل `node index.js` في Termux.
6. سيظهر كود الربط إذا كان الرقم جديدًا.

## ما تم إصلاحه

- لا يعتمد البحث على RPC `find_company_fuzzy` فقط، لأن الخطأ الشائع هو عدم تفعيل `pg_trgm` في Supabase.
- يتم جلب الشركات مباشرة والبحث محليًا بتطبيع عربي قوي.
- عند عدم العثور على شركة يتم إرسال رد واتساب للمرسل، وليس الاكتفاء بالطباعة في Termux.
- يتم تسجيل كل الاستعلامات في `bot_logs`.
